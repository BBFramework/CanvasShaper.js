Carem_SymbolLine

Giới thiệu

Carem_SymbolLine là class dùng để chứa các thuộc tính và phương thức quản lý, và thao tác các dữ liệu liên quan đến đường thẳng.

Thuộc tính

DOMCanvasElement

DOMCanvasElement là thành phần dom trong html5 được lưu trong thành phần Canvas. Tại đây nó được share ra cho Carem_SymbolLine.

Cách truy cập vào DOMCanvasElement như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

console.log(symbol.DOMCanvasElement);

context

context là bảng vẽ được lưu trong thành phần Canvas. Tại đây nó được share ra cho Carem_SymbolLine

Cách truy cập vào context như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

console.log(symbol.context);

Graphics

Graphics lưu trữ đối tượng graphics dùng để xử lý đồ họa, hình họa cho symbol.

Cách truy cập vào Graphics như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

console.log(symbol.Graphics);

SceneObject

SceneObject lưu trữ đối tượng SceneObject dùng để xử lý cách thức làm việc của symbol trên bảng vẽ.

Cách truy cập vào SceneObject như ví dụ sau:

pointArray

pointArray lưu trữ đối tượng pointArray dùng để chứa điểm đầu và điểm cuối của đường thẳng.

Cách truy cập vào pointArray như ví dụ sau:

Phương thức

pushPoint(Point)

pushPoint() được dùng để nạp tọa độ điểm vào trong pointArray.

Không khuyến khích sử dụng phương thức pushPoint

Tham số:
  • Point - Một đối tượng json {x: posX, y: posY} tượng trưng cho tọa độ điểm của điểm đầu hay điểm cuối.

Cách sử dụng pushPoint như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.pushPoint(100);

setStartPoint(x, y)

setStartPoint() được dùng để nạp tọa độ điểm bắt đầu đường thẳng vào trong pointArray.

Tham số:
  • x - Kiểu số (int|float) tượng trưng cho tọa độ x của điểm vị trí điểm bắt đầu.
  • y - Kiểu số (int|float) tượng trưng cho tọa độ y của điểm vị trí điểm bắt đầu.

Cách sử dụng setStartPoint như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.setStartPoint(10, 10);

setEndPoint(x, y)

setEndPoint() được dùng để nạp tọa độ điểm kết thúc đường thẳng vào trong pointArray.

Tham số:
  • x - Kiểu số (int|float) tượng trưng cho tọa độ x của điểm vị trí điểm kết thúc.
  • y - Kiểu số (int|float) tượng trưng cho tọa độ y của điểm vị trí điểm kết thúc.

Cách sử dụng setEndPoint như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.setEndPoint(10, 10);

setOrgin(x, y)

setOrgin() được dùng để cài đặt tọa độ điểm gốc của hình vẽ.

Tham số:
  • x - Kiếu số (int) tượng trưng cho tọa độ x của tọa độ điểm gốc.
  • y - Kiểu số (int) tượng trưng cho tọa độ y của tọa độ điểm gốc.

Cách sử dụng setOrgin() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.setStartPoint(100, 100);

symbol.setEndPoint(200, 100);

symbol.setOrgin(10, 10);

resetOrgin()

resetOrgin() được dùng để khôi phục cài đặt tọa độ điểm gốc của hình vẽ về cài đặt ban đầu.

Cách sử dụng resetOrgin() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.setStartPoint(100, 100);

symbol.setEndPoint(200, 100);

symbol.setOrgin(10, 10); // Current Orgin is (10, 10)

symbol.resetOrgin(); // Current Orgin is (0, 0)

getPosition() được dùng để xem tọa độ điểm vị trí của hình vẽ.

Phương thức getPosition sẽ trả lại một json {x: posX, y: posY} tọa độ điểm của vị trí hình vẽ.

Cách sử dụng getPosition() như ví dụ sau:

var symbol = new Carem_SymbolLine(Layer);

symbol.setStartPoint(100, 100);

symbol.setEndPoint(200, 100);

console.log(symbol.getPosition()); // Output {x: 0, y: 0}

hitPoint(x, y, callback)

hitPoint() được dùng để cài đặt sự kiện chạm hitpoint của hình vẽ.

Tham số:
  • x - Kiếu số (int) tượng trưng cho tọa độ x của vị trí điểm hitpoint.
  • y - Kiểu số (int) tượng trưng cho tọa độ y của vị trí điểm hitpoint.
  • callback - Hàm thực thi khi hình vẽ thực sự bao quanh hitpoint.

Cách sử dụng hitPoint() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.setStartPoint(100, 100);

symbol.setEndPoint(200, 100);

symbol.hitPoint(120, 100, function() {
    console.log("OK");
});

setHitPointTest(callback)

setHitPointTest() được dùng để cài đặt, chỉnh sửa hàm thực thi khi có sự kiện chạm hitpoint của hình vẽ.

Tham số:
  • callback - Hàm thực thi khi hình vẽ thực sự bao quanh hitpoint.

Cách sử dụng setHitPointTest() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.setStartPoint(100, 100);

symbol.setEndPoint(200, 100);

symbol.hitPoint(10, 10, function() {
    console.log("OK");
});

symbol.setHitPointTest(function() {
    console.log("OK! Changed handle.");
});

isDisable()

isDisable() được dùng để kiểm tra xem trạng thái vô hiệu hóa với các sự kiện "mouseup | mouseover | click | hitpoint" mà người ta đã cài đặt trước đó lên hình vẽ.

Mặc định là trạng thái Enable của hình vẽ là true.

Cách sử dụng isDisable() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

console.log(symbol.isDisable()); // Output false

disable()

disable() được dùng để vô hiệu hóa hình vẽ với các sự kiện "mouseup | mouseover | click | hitpoint" mà người ta đã cài đặt trước đó lên hình vẽ.

var Layer = new Carem_Layer("canvas_id");

isHide()

isHide() được dùng để kiểm tra xem trạng thái ẩn của hình vẽ.

Mặc định trạng thái ẩn của hình vẽ là giá trị false.

Cách sử dụng isHide() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

console.log(symbol.isHide()); // Output false

hide()

hide() được dùng để cài đặt trạng thái ẩn cho hình vẽ.

Khi đối tượng được cài đặt trạng thái ẩn, có hai thứ sẽ xãy ra hình vẽ, thứ nhất là hình vẽ sẽ không nhìn thấy được, thứ hai là hình vẽ sẽ không còn hiệu lực với các sự kiện "mouseup | mouseover | click | hitpoint".

Cách sử dụng hide() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.hide();

console.log(symbol.isHide()); // Output true

isShow()

isShow() được dùng để kiểm tra xem trạng thái hiển thị của hình vẽ.

Mặc định trạng thái hiển thị của hình vẽ là giá trị true.

Cách sử dụng isShow() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

show()

show() được dùng để khôi phục cài đặt trạng thái hiển thị của hình vẽ về cài đặt gốc.

Khi đối tượng được cài đặt trạng thái ẩn, có hai thứ sẽ xãy ra hình vẽ, thứ nhất là hình vẽ sẽ không nhìn thấy được, thứ hai là hình vẽ sẽ không còn hiệu lực với các sự kiện "mouseup | mouseover | click | hitpoint", và sau đó người ta có thể sử dụng phương thức show để khôi phục cài đặt từ trạng thái ẩn này về trạng thái hiển thị.

Cách sử dụng show() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.hide();

symbol.show();

console.log(symbol.isHide()); // Output false

invisible()

invisible() được dùng để cài đặt trạng thái hiển thị của hình vẽ sang trạng thái không hiển thị.

Mặc định hình vẽ có trạng thái nhìn thấy được, và sau đó người ta sử dụng phương thức invisible để chuyển hình vẽ về trạng thái không nhìn thấy được.

Chú ý, phương thức invisible hoàn toàn khác với phương thức hide, phương thức invisible chỉ làm cho hình vẽ vô hình, trong khi nó vẫn còn hiệu lực với các sự kiện "mouseup | mouseover | click | hitpoint".

Cách sử dụng invisible() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.invisible();

Tham số:

var Layer = new Carem_Layer("canvas_id");

setStrokeColor(color, startPoint, endPoint, radius)

setStrokeColor() được dùng để cài đặt màu viền cho hình vẽ.

Tham số:
  • color - Màu của nền (solid | gradient).
  • startPoint - Điểm bắt đầu tô màu của các màu có trạng thái chuyển sắc (gradient).
  • endPoint - Điểm kết thúc tô màu của các màu có trạng thái chuyển sắc (gradient).
  • radius - Bán kính vòng cung của màu tô (radius gradient).

Cách sử dụng setStrokeColor() với màu nguyên như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolCircle(Layer);

symbol.setStrokeColor([{r:255, g:0, b:0, a:1, stop:0}]);// red fill

Cách sử dụng setStrokeColor() với màu linear gradient như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolCircle(Layer);

symbol.setStrokeColor(
    [
        {r:255, g:0, b:0, a:1, stop:0}, // red color
        {r:0, g:255, b:0, a:1, stop:0.5}, // green color
        {r:0, g:0, b:255, a:1, stop:1} // blue color
    ],
    {x: 0, y: 0}, // start point
    {x: 0, y: 100}, // end point
);// [red -> green -> blue] linear gradient fill

Cách sử dụng setStrokeColor() với màu radial gradient như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolCircle(Layer);

symbol.setStrokeColor(
    [
        {r:255, g:0, b:0, a:1, stop:0}, // red color
        {r:0, g:255, b:0, a:1, stop:0.5}, // green color
        {r:0, g:0, b:255, a:1, stop:1} // blue color
    ],
    {x: 0, y: 0}, // start point
    {x: 0, y: 100}, // end point
    20 // radius
);// [red -> green -> blue] radial gradient fill

setStrokeImage(asset, mode)

setStrokeImage() được dùng để cài đặt viền của hình vẽ bằng hình ảnh.

Tham số:
  • asset - Tài nguyên chứa hình ảnh dùng để làm đường viền cho hình vẽ.
  • mode - Chế độ tô.

Tham số mode là chế độ tô được sử dụng với các hằng số sau:

  • CAREM_REPEAT - Mặc định, Chế độ lặp lại lẫn chiều ngang và chiều dọc.
  • CAREM_REPEAT_X - Chế độ lặp lại theo chiều ngang.
  • CAREM_REPEAT_Y - Chế độ lặp lại theo chiều dọc.
  • CAREM_NO_REPEAT - Chế độ tô một lần (không lặp).

Cách sử dụng setStrokeImage() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.setStrokeImage(AssetMng.getAsset("image.jpg"), CAREM_REPEAT); // set transparent to 100.

setStrokeCap(type)

setStrokeCap() được dùng để cài đặt hình dáng của các điểm cuối của đường viền của hình vẽ, hoặt đường thẳng.

Tham số:
  • type - Hình dạng của các điểm cuối của các đường thẳng hay đường viền.

Tham số type được sử dụng với các hằng số sau:

  • CAREM_LINE_CAP_DEFAULT - Mặc định, "butt".
  • CAREM_LINE_CAP_BUTT - "butt"
  • CAREM_LINE_CAP_ROUND - "round".
  • CAREM_LINE_CAP_SQUARE - "square".

Cách sử dụng setStrokeCap() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.setStrokeCap(CAREM_LINE_CAP_BUTT);

setStrokeWidth(value)

setStrokeWidth() được dùng để cài đặt độ dày viền của hình vẽ, hoặt đường thẳng.

Tham số:
  • value - Kiểu số (int|float) tượng trưng cho độ dày của đường viền.

Cách sử dụng setStrokeWidth() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.setStrokeWidth(25);

setShadow(color, offset, blur)

setShadow() được dùng để cài đặt bóng đỗ của hình vẽ.

Tham số:
  • color - Đối tượng json {r: red, g: green, b: blue, a: alpha} tượng trưng cho màu bóng đỗ.
  • offset - Đối tượng json {x: offsetX, y: offsetY} tượng trưng cho khoảng cách của bóng đỗ với hình vẽ.
  • blur - Kiểu số (int|float) tượng trưng cho kích thước của bóng đỗ.

Cách sử dụng setShadow() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.setShadow(
    {r: 128, g: 128, b: 128, a: 1},
    {x: 5, y: 5},
    15
);

clearShadow()

clearShadow() được dùng để cài đặt bóng đỗ của hình vẽ về rỗng.

Cách sử dụng clearShadow() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.setShadow(
    {r: 128, g: 128, b: 128, a: 1},
    {x: 5, y: 5},
    15
);

symbol.clearShadow();

Draw()

Draw() được dùng để vẽ hình vẽ.

Cách sử dụng Draw() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolLine(Layer);

symbol.Draw();

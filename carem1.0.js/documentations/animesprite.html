Carem_AnimeSprite

Giới thiệu

Carem_AnimeSprite là class dùng để chứa các thuộc tính và phương thức của Carem_AnimeSprite.

Thuộc tính

DOMCanvasElement

DOMCanvasElement là thành phần dom trong html5 được lưu trong thành phần Canvas. Tại đây nó được share ra cho Carem_AnimeSprite.

Cách truy cập vào DOMCanvasElement như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var animeSprite = new Carem_AnimeSprite(Layer, AssetMng.getAsset("image.jpg"), 100, 50);

console.log(animeSprite.DOMCanvasElement);

Phương thức

setAction(value)

setAction() được dùng để cài đặt action của Carem_AnimeSprite.

Tham số:
  • value - Kiểu số (int) tượng trưng cho vị trí hàng trong sprite hoạt hình.

Cách sử dụng setAction() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var Player = new Carem_AnimeSprite(Layer, AssetMng.getAsset("image.jpg"), 100, 50);

var Player.Actions = {Xuoi: 0, Nguoc: 1};

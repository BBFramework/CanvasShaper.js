Carem_SymbolRect

Giới thiệu

Carem_SymbolRect là class dùng để chứa các thuộc tính và phương thức quản lý, và thao tác các dữ liệu liên quan đến hình tứ giác.

Thuộc tính

DOMCanvasElement

DOMCanvasElement là thành phần dom trong html5 được lưu trong thành phần Canvas. Tại đây nó được share ra cho Carem_SymbolRect.

Cách truy cập vào DOMCanvasElement như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolRect(Layer);

console.log(symbol.DOMCanvasElement);

context

context là bảng vẽ được lưu trong thành phần Canvas. Tại đây nó được share ra cho Carem_SymbolRect

Cách truy cập vào context như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolRect(Layer);

console.log(symbol.context);

Graphics

Graphics lưu trữ đối tượng graphics dùng để xử lý đồ họa, hình họa cho symbol.

Cách truy cập vào Graphics như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolRect(Layer);

console.log(symbol.Graphics);

SceneObject

SceneObject lưu trữ đối tượng SceneObject dùng để xử lý cách thức làm việc của symbol trên bảng vẽ.

Cách truy cập vào SceneObject như ví dụ sau:

Phương thức

setOrgin(x, y)

setOrgin() được dùng để cài đặt tọa độ điểm gốc của hình vẽ.

Tham số:
  • x - Kiếu số (int) tượng trưng cho tọa độ x của tọa độ điểm gốc.
  • y - Kiểu số (int) tượng trưng cho tọa độ y của tọa độ điểm gốc.

Cách sử dụng setOrgin() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolRect(Layer);

symbol.setOrgin(10, 10);

resetOrgin()

resetOrgin() được dùng để khôi phục cài đặt tọa độ điểm gốc của hình vẽ về cài đặt ban đầu.

Cách sử dụng resetOrgin() như ví dụ sau:

getPosition()

getPosition() được dùng để xem tọa độ điểm vị trí của hình vẽ.

Phương thức getPosition sẽ trả lại một json {x: posX, y: posY} tọa độ điểm của vị trí hình vẽ.

Cách sử dụng getPosition() như ví dụ sau:

hitPoint(x, y, callback)

hitPoint() được dùng để cài đặt sự kiện chạm hitpoint của hình vẽ.

Tham số:
  • x - Kiếu số (int) tượng trưng cho tọa độ x của vị trí điểm hitpoint.
  • y - Kiểu số (int) tượng trưng cho tọa độ y của vị trí điểm hitpoint.
  • callback - Hàm thực thi khi hình vẽ thực sự bao quanh hitpoint.

Cách sử dụng hitPoint() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolRect(Layer);

symbol.setSize(100, 100);

symbol.hitPoint(10, 10, function() {
    console.log("OK");
});

setHitPointTest(callback)

setHitPointTest() được dùng để cài đặt, chỉnh sửa hàm thực thi khi có sự kiện chạm hitpoint của hình vẽ.

Tham số:
  • callback - Hàm thực thi khi hình vẽ thực sự bao quanh hitpoint.

Cách sử dụng setHitPointTest() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolRect(Layer);

symbol.setSize(100, 100);

symbol.hitPoint(10, 10, function() {
    console.log("OK");
});

symbol.setHitPointTest(function() {
    console.log("OK! Changed handle.");
});

isHide()

isHide() được dùng để kiểm tra xem trạng thái ẩn của hình vẽ.

Mặc định trạng thái ẩn của hình vẽ là giá trị false.

Cách sử dụng isHide() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolRect(Layer);

hide()

hide() được dùng để cài đặt trạng thái ẩn cho hình vẽ.

Khi đối tượng được cài đặt trạng thái ẩn, có hai thứ sẽ xãy ra hình vẽ, thứ nhất là hình vẽ sẽ không nhìn thấy được, thứ hai là hình vẽ sẽ không còn hiệu lực với các sự kiện "mouseup | mouseover | click | hitpoint".

Cách sử dụng hide() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolRect(Layer);

symbol.hide();

console.log(symbol.isHide()); // Output true

isShow()

isShow() được dùng để kiểm tra xem trạng thái hiển thị của hình vẽ.

Mặc định trạng thái hiển thị của hình vẽ là giá trị true.

show()

show() được dùng để khôi phục cài đặt trạng thái hiển thị của hình vẽ về cài đặt gốc.

Khi đối tượng được cài đặt trạng thái ẩn, có hai thứ sẽ xãy ra hình vẽ, thứ nhất là hình vẽ sẽ không nhìn thấy được, thứ hai là hình vẽ sẽ không còn hiệu lực với các sự kiện "mouseup | mouseover | click | hitpoint", và sau đó người ta có thể sử dụng phương thức show để khôi phục cài đặt từ trạng thái ẩn này về trạng thái hiển thị.

Cách sử dụng show() như ví dụ sau:

invisible()

invisible() được dùng để cài đặt trạng thái hiển thị của hình vẽ sang trạng thái không hiển thị.

Mặc định hình vẽ có trạng thái nhìn thấy được, và sau đó người ta sử dụng phương thức invisible để chuyển hình vẽ về trạng thái không nhìn thấy được.

Chú ý, phương thức invisible hoàn toàn khác với phương thức hide, phương thức invisible chỉ làm cho hình vẽ vô hình, trong khi nó vẫn còn hiệu lực với các sự kiện "mouseup | mouseover | click | hitpoint".

Cách sử dụng invisible() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolRect(Layer);

symbol.invisible();

setBackground(color, startPoint, endPoint, radius)

setBackground() được dùng để cài đặt màu nền cho hình vẽ.

Tham số:
  • color - Màu của nền (solid | gradient).
  • startPoint - Điểm bắt đầu tô màu của các màu có trạng thái chuyển sắc (gradient).
  • endPoint - Điểm kết thúc tô màu của các màu có trạng thái chuyển sắc (gradient).
  • radius - Bán kính vòng cung của màu tô (radius gradient).

Cách sử dụng setBackground() với màu nguyên như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolRect(Layer);

symbol.setBackground([{r:255, g:0, b:0, a:1, stop:0}]);// red fill

Cách sử dụng setBackground() với màu linear gradient như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolRect(Layer);

symbol.setBackground(
    [
        {r:255, g:0, b:0, a:1, stop:0}, // red color
        {r:0, g:255, b:0, a:1, stop:0.5}, // green color
        {r:0, g:0, b:255, a:1, stop:1} // blue color
    ],
    {x: 0, y: 0}, // start point
    {x: 0, y: 100}, // end point
);// [red -> green -> blue] linear gradient fill

Cách sử dụng setBackground() với màu radial gradient như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

var symbol = new Carem_SymbolRect(Layer);

symbol.setBackground(
    [
        {r:255, g:0, b:0, a:1, stop:0}, // red color
        {r:0, g:255, b:0, a:1, stop:0.5}, // green color
        {r:0, g:0, b:255, a:1, stop:1} // blue color
    ],
    {x: 0, y: 0}, // start point
    {x: 0, y: 100}, // end point
    20 // radius
);// [red -> green -> blue] radial gradient fill

setBackgroundImage(asset, mode)

setBackgroundImage() được dùng để cài đặt hình nền của hình vẽ bằng hình ảnh.

Tham số:
  • asset - Tài nguyên chứa hình ảnh dùng để làm hình nền cho hình vẽ.
  • mode - Chế độ tô.

Tham số mode là chế độ tô được sử dụng với các hằng số sau:

  • CAREM_REPEAT - Mặc định, Chế độ lặp lại lẫn chiều ngang và chiều dọc.
  • CAREM_REPEAT_X - Chế độ lặp lại theo chiều ngang.
  • CAREM_REPEAT_Y - Chế độ lặp lại theo chiều dọc.
  • CAREM_NO_REPEAT - Chế độ tô một lần (không lặp).

Cách sử dụng setBackgroundImage() như ví dụ sau:

var Layer = new Carem_Layer("canvas_id");

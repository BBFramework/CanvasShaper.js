Carem_Math

Giới thiệu

Carem_Math là class dùng để chứa các phương thức toán học, Carem_Math là class tĩnh, bạn có thể sử dụng trực tiếp mà không cần tạo cụ thể.

Phương thức

Min(value1, value2)

Min() sẽ so sánh và lấy ra cho bạn số nhỏ nhất trong hai số mà bạn nhập vào tham số value1 và value2.

Các tham số:
  • Value1 - Giá trị cần so sánh.
  • Value2 - Giá trị cần so sánh.

Cách sử dụng Min() như ví dụ sau:

var number = Carem_Math.Min(100, 50);

console.log(number); // number is save 50

Max(value1, value2)

Max() sẽ so sánh và lấy ra cho bạn số lớn nhất trong hai số mà bạn nhập vào tham số value1 và value2.

Các tham số:
  • Value1 - Giá trị cần so sánh.
  • Value2 - Giá trị cần so sánh.

Cách sử dụng Max() như ví dụ sau:

var number = Carem_Math.Max(100, 50);

console.log(number); // number is save 100

RadianFromAngle(Angle)

RadianFromAngle() sẽ giúp chuyển đổi từ góc độ sang radian.

Các tham số:
  • Angle - Góc độ (int) mà muốn chuyển đổi thành radian

Cách sử dụng RadianFromAngle() như ví dụ sau:

var number = Carem_Math.RadianFromAngle(180);

console.log(number); // number is save 3.14

AngleFromRadian(Radian)

AngleFromRadian() sẽ giúp chuyển đổi từ radian sang góc độ.

Các tham số:
  • Radian - Radian (int | float) mà muốn chuyển đổi thành góc độ.

Cách sử dụng AngleFromRadian() như ví dụ sau:

var number = Carem_Math.AngleFromRadian(3.14);

console.log(number); // number is save 180

VectorFromAngle(Angle)

VectorFromAngle() sẽ giúp chuyển đổi từ góc độ sang vector.

Các tham số:
  • Angle - Góc độ (int | float) mà muốn chuyển đổi thành vector.

Trả lại: Đối tượng json {x: value, y: value}

Cách sử dụng VectorFromAngle() như ví dụ sau:

var vector = Carem_Math.VectorFromAngle(180);

console.log(vector); // vector is {x: -1, y: 0}

VectorFromRadian(Radian)

VectorFromRadian() sẽ giúp chuyển đổi từ radian sang vector.

Các tham số:
  • Radian - Radian (int | float) mà muốn chuyển đổi thành vector.

Trả lại: Đối tượng json {x: value, y: value}

Cách sử dụng VectorFromRadian() như ví dụ sau:

var vector = Carem_Math.VectorFromRadian(3.14);

console.log(vector); // vector is {x: -1, y: 0}

Random(min, max)

Random() sẽ giúp tạo số mặc định trong giới hạn từ min đến max.

Các tham số:
  • min - là chỉ số nhỏ nhất.
  • max - là chỉ số lớn nhất.

Cách sử dụng Random() như ví dụ sau:

var number = Carem_Math.Random(10, 20);

console.log(number); // Maybe number is 10, and it never less then 9 or greater 21

intRandom(min, max)

intRandom() sẽ giúp tạo số mặc định trong giới hạn từ min đến max.

Các tham số:
  • min - là chỉ số nhỏ nhất.
  • max - là chỉ số lớn nhất.

Cách sử dụng intRandom() như ví dụ sau:

console.log(number); // Maybe number is 10, and it never less then 9 or greater 21

Carem_Filter

Giới thiệu

Carem_Filter là class dùng để chứa các phương thức chỉnh sửa hình ảnh, Carem_Filter là class tĩnh, bạn có thể sử dụng trực tiếp mà không cần tạo cụ thể.

Phương thức

GrayScale(pixels)

GrayScale() sẽ filter dữ liệu trên pixels hình ảnh thành màu trắng đen.

Các tham số:
  • pixels - Dữ liệu màu của hình ảnh trên pixels.

Cách sử dụng GrayScale() như ví dụ sau:

var Layer = new Carem_Layer();

var Image = new Carem_Image(Layer, Asset)

Image.setFilter(Carem_Filter.GrayScale);

Brightness(pixels, amount)

Brightness() sẽ blend màu hình ảnh sáng hơn.

Các tham số:
  • pixels - Dữ liệu màu của hình ảnh trên pixels.
  • amount - Độ sáng cần thiết.

Cách sử dụng Brightness() như ví dụ sau:

var Layer = new Carem_Layer();

var Image = new Carem_Image(Layer, Asset)

Image.setFilter(Carem_Filter.Brightness, 100);

Threshold(pixels, amount)

Threshold() sẽ filter dữ liệu màu hình ảnh thành trắng đen.

Các tham số:
  • pixels - Dữ liệu màu của hình ảnh trên pixels.
  • amount - Độ sáng cần thiết.

Cách sử dụng Threshold() như ví dụ sau:

var Layer = new Carem_Layer();

var Image = new Carem_Image(Layer, Asset)

Image.setFilter(Carem_Filter.Threshold, 100);

Convolute(pixels, weight, opaque)

Convolute() sẽ Noise, Sharp các dữ liệu màu hình ảnh trên pixel.

Các tham số:
  • pixels - Dữ liệu màu của hình ảnh trên pixels.
  • weight - Mức độ biến đổi.
  • opaque - Độ chắn sáng.

Cách sử dụng Convolute() như ví dụ sau:

var Layer = new Carem_Layer();

var Image = new Carem_Image(Layer, Asset)

Image.setFilter(Carem_Filter.Convolute, [0, -1, 0,-1, 5, -1, 0, -1, 0]);

ConvoluteFloat32(pixels, weight, opaque)

ConvoluteFloat32() sẽ Noise, Sharp các dữ liệu màu hình ảnh trên pixel.

Các tham số:
  • pixels - Dữ liệu màu của hình ảnh trên pixels.
  • weight - Mức độ biến đổi.
  • opaque - Độ chắn sáng.

Cách sử dụng ConvoluteFloat32() như ví dụ sau:

var Layer = new Carem_Layer();

var Image = new Carem_Image(Layer, Asset)

Image.setFilter(Carem_Filter.ConvoluteFloat32, [0, -1, 0,-1, 5, -1, 0, -1, 0]);

Carem.Button

Giới thiệu

Carem.Button là class dùng để chứa các phương thức toán học, Carem.Button là class tĩnh, bạn có thể sử dụng trực tiếp mà không cần tạo cụ thể.

Thuộc tính

DOMCanvasElement

DOMCanvasElement là thành phần dom trong html5 được lưu trong thành phần Canvas. Tại đây nó được share ra cho Carem.Button.

Cách truy cập vào DOMCanvasElement như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

var button = new Carem.Button(Layer, AssetMng.getAsset("image.jpg"), 20, 20, 100, 50);

console.log(button.DOMCanvasElement);

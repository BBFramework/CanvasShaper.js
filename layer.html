Carem.Layer

Giới thiệu

Class Layer chứa các thuộc tính và phương thức của layer. Layer trong Carem giúp phát triển viên quản lý các tài nguyên của game theo chế độ layer. Tức là Layer chứa và hiển thị các hình ảnh, các hoạt hình, hiệu ứng particle.

Thuộc tính

DOMElement

DOMElement là thành phần dom của canvas.

Bạn có thể truy cập vào thành phần dom của canvas bằng như cách sau:

var Layer = new Carem.Layer("canvas_id");

console.log(Layer.DOMElement);

console.log(Layer.getElement());

console.log(Layer.getDOM());

context

context là thành phần context của canvas. Nó được dùng để vẽ đồ họa lên canvas.

Bạn có thể truy cập vào thành phần context của layer bằng như cách sau:

var Layer = new Carem.Layer("canvas_id");

console.log(Layer.context);

console.log(Layer.getContext());

width

width là thành phần nắm giữ thông số độ rộng của canvas.

Bạn có thể truy cập vào thành phần width của layer bằng như cách sau:

var Layer = new Carem.Layer("canvas_id");

console.log(Layer.width);

console.log(Layer.getWidth());

height

height là thành phần nắm giữ thông số độ cao của canvas.

Bạn có thể truy cập vào thành phần height của layer bằng như cách sau:

var Layer = new Carem.Layer("canvas_id");

console.log(Layer.height);

console.log(Layer.getHeight());

childList

childList là thành phần nắm giữ các thành phần đồ họa của canvas. Nhờ việc lưu trữ vào một nơi tập trung là childList, chúng sẽ dễ dàng trong quản lý và vẽ lên canvas.

Bạn có thể truy cập vào thành phần childList của layer bằng như cách sau:

var Layer = new Carem.Layer("canvas_id");

console.log(Layer.childList);

console.log(Layer.getChilds());

buttonList

buttonList là thành phần nắm giữ các nút nhấn được tạo từ Carem.Button của canvas. Nhờ việc lưu trữ vào một nơi tập trung là buttonList, chúng sẽ dễ dàng trong quản lý và áp dụng các sự kiện "mouseup | mouseover | click".

Bạn có thể truy cập vào thành phần buttonList của layer bằng như cách sau:

var Layer = new Carem.Layer("canvas_id");

console.log(Layer.buttonList);

console.log(Layer.getButtons());

Phương thức

getChilds()

getChilds() sẽ lấy các thành phần đồ họa trên layer, tập hợp chúng lại và trả về cho bạn một mảng chứa các thành phần đồ họa đó.

Cách sử dụng getChilds() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

// Add Childs Code...

console.log(Layer.getChilds());

getButtons()

getButtons() sẽ lấy các thành phần nút nhấn trên layer, tập hợp chúng lại và trả về cho bạn một mảng chứa các thành phần nút nhấn đó.

Cách sử dụng getButtons() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

// Add Buttons Code...

console.log(Layer.getButtons());

getElement()

getElement() sẽ trả lại thành phần DOM của canvas trong html5.

Cách sử dụng getElement() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

console.log(Layer.getElement());

getDOM()

getDOM() sẽ trả lại thành phần DOM của canvas trong html5.

Cách sử dụng getDOM() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

console.log(Layer.getDOM());

getWidth()

getWidth() sẽ trả lại độ rộng của canvas trong html5.

Cách sử dụng getWidth() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

console.log(Layer.getWidth());

getHeight()

getHeight() sẽ trả lại độ cao của canvas trong html5.

Cách sử dụng getHeight() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

console.log(Layer.getHeight());

getContext()

getContext() sẽ trả lại context đồ họa của canvas trong html5. Context của canvas chủ yếu được dùng để vẽ các hình họa và hình ảnh.

Cách sử dụng getContext() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

console.log(Layer.getContext());

setClearColor(r, g, b, a)

setClearColor() dùng để cài đặt màu clear của layer mỗi khi refresh lại view của nó trên canvas.

Các tham số:
  • r - Tham số cho kênh màu đỏ
  • g - Tham số cho kênh màu lục
  • b - Tham số cho kênh màu lam
  • a - Tham số cho độ trong suốt

Cách sử dụng setClearColor() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

Layer.setClearColor(255, 255, 255, 1.0); // white color

setBackground(r, g, b, a)

setBackground() dùng để cài đặt màu clear của layer mỗi khi refresh lại view của nó trên canvas.

Các tham số:
  • r - Tham số cho kênh màu đỏ
  • g - Tham số cho kênh màu lục
  • b - Tham số cho kênh màu lam
  • a - Tham số cho độ trong suốt

Cách sử dụng setBackground() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

Layer.setBackground(255, 255, 255, 1.0); // white color

clear()

clear() dùng để xóa hoặt tô màu cho background của layer.

Cách sử dụng clear() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

Layer.clear();

addChild(child)

addChild() dùng để thêm các thành phần đồ vào trong layer. Thực tế chúng được đặt một cách tuần tự vào trong một mảng childList trong đối tượng Layer.

Các tham số:
  • child - Child là các thành phần đồ họa.

Cách sử dụng addChild() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

var Image = new Carem.Image(Layer, Asset); // Added Child Into Layer.

var Layer02 = new Carem.Layer("canvas_id");

Layer02.addChild(Image)

addButton(child)

addButton() dùng để thêm các nút nhấn vào trong layer. Thực tế chúng được đặt một cách tuần tự vào trong một mảng buttonList trong đối tượng Layer.

Các tham số:
  • child - Child là các thành phần nút nhấn.

Cách sử dụng addChild() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

var button = new Carem.Button(Layer, Asset, 100, 50); // Added Child Into Layer.

var Layer02 = new Carem.Layer("canvas_id");

Layer02.addChild(button)

Swap()

Swap() dùng để vẽ các thành phần đồ họa được đặt trong mảng childList của đổi tượng layer.

Cách sử dụng Swap() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

// ... Add childs.

Layer.Swap();

addEventListener(eventType, callback, track)

addEventListener() dùng để thêm vào các sự kiện từ các input như Mouse, Keyboard, v.v... Nó giống với cách sử dụng của addEventListener trong javascript của các thành phần html5.

Cách sử dụng addEventListener() như ví dụ sau:

var Layer = new Carem.Layer("canvas_id");

// ... Add childs.

Layer.addEventListenter("click", function(e) {
... Code will do on fire the "click" event.
}, false);

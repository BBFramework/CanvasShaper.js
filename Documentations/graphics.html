CanvasShaper_Graphics

Giới thiệu

CanvasShaper_Graphics là class dùng để chứa các thuộc tính và phương thức quản lý, và thao tác đồ họa. Các kiểu hiển thị (Styling) của hình vẽ (tức là các thành phần đồ họa, đặc biệt là các thành phần hình họa, trong CanvasShaper gọi là Symbol) được điều khiển qua CanvasShaper_Graphics.

Thuộc tính

context

context là một đối tượng bảng vẽ của canvas.

Cách truy cập vào context như ví dụ sau:

var Graphics = new CanvasShaper_Graphics(canvas_context);

console.log(Graphics.context);

patternFill

patternFill là một thành phần chứa mẫu tô lên hình vẽ trên bảng vẽ canvas.

Cách truy cập vào patternFill như ví dụ sau:

var Graphics = new CanvasShaper_Graphics(canvas_context);

console.log(Graphics.patternFill);

gradientFill là một thành phần chứa mẫu tô với chuyển sắc lên hình vẽ trên bảng vẽ canvas.

Cách truy cập vào gradientFill như ví dụ sau:

var Graphics = new CanvasShaper_Graphics(canvas_context);

Phương thức

setMask()

setMask() sẽ cài đặt bảng vẽ trong trạng thái mask.

Khi tất cả các thành phần đồ họa được vẽ sau khi cài đặt trạng thái mask, chúng sẽ ẩn sau thành phần đồ họa mà được cài đặt mask.

Cách sử dụng setMask() như ví dụ sau:

var rect = new CanvasShaper_SymbolRect(Layer);

rect.setMask();

var round_rect = new CanvasShaper_SymbolRoundRect(Layer);

var arc = new CanvasShaper_SymbolArc(Layer);

arc.releaseMask();

// Result: 'rect' will mask for 'round_rect', 'arc' will draw normal.

clearMask()

clearMask() sẽ hủy trạng thái mask cho bảng vẽ. Hủy ở đây được hiểu là, dừng thôi không mask nữa chứ không phải là thoát ra, hay kết thúc mask.

Cách sử dụng clearMask() như ví dụ sau:

var rect = new CanvasShaper_SymbolRect(Layer);

rect.setMask();

var round_rect = new CanvasShaper_SymbolRoundRect(Layer);

var arc = new CanvasShaper_SymbolArc(Layer);

rect.clearMask();

// Result: 'rect', 'round_rect' and 'arc' will draw normal.

endMask()

endMask() sẽ kết thúc trạng thái mask trên bản vẽ.

Khi đối tượng được cài đặt kết thúc trạng thái mask, thì nó sẽ vẫn được vẽ như bình thường.

Cách sử dụng endMask() như ví dụ sau:

var rect = new CanvasShaper_SymbolRect(Layer);

rect.setMask();

var round_rect = new CanvasShaper_SymbolRoundRect(Layer);

var arc = new CanvasShaper_SymbolArc(Layer);

arc.endMask();

// Result: 'rect' will mask for 'round_rect', 'arc' will draw normal.

releaseMask()

releaseMask() sẽ kết thúc trạng thái mask trên bản vẽ.

Khi đối tượng được cài đặt kết thúc trạng thái mask, thì nó sẽ vẫn được vẽ như bình thường.

Cách sử dụng releaseMask() như ví dụ sau:

var rect = new CanvasShaper_SymbolRect(Layer);

rect.setMask();

var round_rect = new CanvasShaper_SymbolRoundRect(Layer);

var arc = new CanvasShaper_SymbolArc(Layer);

arc.releaseMask();

// Result: 'rect' will mask for 'round_rect', 'arc' will draw normal.

setSave(State)

setSave() được dùng để cài đặt thành phần save.

Tham số:
  • State - Giá trị bool mà được gán vào thành phần save.

Nếu đưa vào giá trị true, thì thành phần Graphics.save sẽ được gắn với giá trị true, tương tự với giá trị false

Cách sử dụng setSave() như ví dụ sau:

var Graphics = new CanvasShaper_Graphics(canvas_context);

Graphics.setSave(true);

Cách sử dụng setRestore() như ví dụ sau:

var Graphics = new CanvasShaper_Graphics(canvas_context);

Graphics.setRestore(true);

drawImage(Asset, x, y)

drawImage() được dùng để vẽ một hình ảnh lên trên bảng vẽ.

Tham số:
  • Asset - Asset mà được lấy từ trong AssetManager.
  • x - Vị trí x mà hình ảnh được vẽ lên.
  • y - Vị trí y mà hình ảnh được vẽ lên.

Cách sử dụng drawImage() như ví dụ sau:

var AssetMng = new CanvasShaper_AssetManager(".jpg|.gif|.png");

var Graphics = new CanvasShaper_Graphics(Canvas);

AssetMng.QueueFile("File.jpg");

Graphics.drawImage(AssetMng.getAsset("File.jpg"), 100, 100);

getImageData(imgData)

getImageData() sẽ trả đối tượng dữ liệu hình ảnh.

Tham số:
  • imgData - Dữ liệu của hình ảnh.

Trả lại: Đối tượng json dữ liệu hình ảnh {Data: imgData, Width: imgWidth, Height: imgHeight}.

var Graphics = new CanvasShaper_Graphics(Canvas);

var imgData = Graphics.copyImageData(100, 100, 200, 100);

var Image = Graphics.getImageData(imgData);

copyImageData(x, y, width, height)

copyImageData() sẽ sao chép dữ liệu pixel trên bảng vẽ.

Tham số:
  • x - Vị trí x điểm bắt đầu sao chép.
  • y - Vị trí y điểm bắt đầu sao chép.
  • width - Chiều rộng (số pixels theo chiều ngang) bắt đầu từ vị trí x điểm bắt đầu.
  • height - Chiều cao (số pixels theo chiều dọc) bắt đầu từ vị trí y điểm bắt đầu.

Trả lại: đối tượng chứa width, height, và dữ liệu hình ảnh {data: imgdata, width: imgWidth, height: imgHeight}

Cách sử dụng copyImageData() như ví dụ sau:

var Graphics = new CanvasShaper_Graphics(Canvas);

var imgData = Graphics.copyImageData(100, 100, 200, 100);

pasteImageData(imgData, x, y)

pasteImageData() sẽ vẽ lại dữ liệu pixel trên bảng vẽ.

Tham số:
  • imgData - Dữ liệu pixels đã sao chép.
  • x - Vị trí x điểm bắt đầu vẽ lại.
  • y - Vị trí y điểm bắt đầu vẽ lại.

Cách sử dụng pasteImageData() như ví dụ sau:

var Graphics = new CanvasShaper_Graphics(Canvas);

var imgData = Graphics.copyImageData(100, 100, 200, 100);

Graphics.pasteImageData(imgData, 100, 100);

createPattern(asset, mode)

createPattern() được dùng để tạo ra các pattern trong việc tô màu hay tô viền cho hình vẽ.

Tham số:
  • asset - Tài liệu hình ảnh được dùng để tạo pattern.
  • mode - Chế độ repeat.

Tham số mode được dùng với các hằng số sau:

  • CANVAS_SHAPER_REPEAT - Lặp đều từ ngang đến dọc.
  • CANVAS_SHAPER_REPEAT_X - Lặp đều chỉ theo chiều ngang.
  • CANVAS_SHAPER_REPEAT_Y - Lặp đều chỉ theo chiều dọc.
  • CANVAS_SHAPER_NO_REPEAT - Chỉ vẽ một lần duy nhất.

Cách sử dụng createPattern() như ví dụ sau:

var Graphics = new CanvasShaper_Graphics(Canvas);

Graphics.createPattern(AssetMng.getAsset("image.jpg"), CANVAS_SHAPER_REPEAT);

  • endPoint - Đối tượng gồm {x, y} chứa tọa độ điểm kết thúc chuyển sắc.
  • radius - Bán kính của vòng tròn chuyển sắc.

Cách sử dụng setStrokeColor() như ví dụ sau:

var Graphics = new CanvasShaper_Graphics(Canvas);

Graphics.setStrokeColor(
    [
        {r:255, g:255, b:255, a:0.5, stop:0},
        {r:128, g:128, b:128, a:0.5, stop:1}
    ],
    {x: 0, y:0}, {x: 0, y:100}, 50
);

setPatternStroke(asset, mode)

setPatternStroke() được dùng để cài đặt các pattern trong việc tô viền cho hình vẽ.

Tham số:
  • asset - Tài liệu hình ảnh được dùng để tạo pattern.
  • mode - Chế độ repeat.

Tham số mode được dùng với các hằng số sau:

  • CANVAS_SHAPER_REPEAT - Lặp đều từ ngang đến dọc.
  • CANVAS_SHAPER_REPEAT_X - Lặp đều chỉ theo chiều ngang.
  • CANVAS_SHAPER_REPEAT_Y - Lặp đều chỉ theo chiều dọc.
  • CANVAS_SHAPER_NO_REPEAT - Chỉ vẽ một lần duy nhất.

Cách sử dụng setPatternStroke() như ví dụ sau:

var Graphics = new CanvasShaper_Graphics(Canvas);

Graphics.setPatternStroke(AssetMng.getAsset("image.jpg"), CANVAS_SHAPER_REPEAT);

Tham số lineCap được dùng với các hằng số

Tham số lineJoin được dùng với các hằng số

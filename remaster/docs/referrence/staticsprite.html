Carem.StaticSprite

Giới thiệu

Carem.StaticSprite là class dùng để chứa các thành phần dữ liệu, các hàm phương thức nhằm quản lý, và thao tác các dữ liệu liên quan đến sprite tĩnh.

Carem.StaticSprite được mở rộng với Carem.Graphics, Carem.SceneObject, Carem.Physics, Carem.WorldLimit, và Carem.Collision.

Thành phần dữ liệu

DOMCanvasElement

Được dùng để chứa thành phần dom trong html5 được lưu trong thành phần canvas. Tại đây nó được share ra cho Carem.StaticSprite.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var sprite = new Carem.StaticSprite( layer, assetMgr.GetAsset( 'image.jpg' ), 20, 20, 100, 50 );
  • console.log( sprite.DOMCanvasElement );

context

Được dùng để chứa thành phần drawing context của canvas.

Cách sử dụng

  • console.log( sprite.context );

image

Được dùng để lưu trữ tài liệu hình ảnh của sprite.

Thành phần constructor

Carem.StaticSprite( layer, asset, x, y, width, height )

Tạo mới một đối tượng Carem.StaticSprite.

Tham số

  • layer:Carem.Layer Đối tượng layer dùng render hình vẽ.
  • asset:any Đối tượng tài liệu hình ảnh.
  • x:number Tọa độ x của điểm bắt đầu cắt ảnh.
  • y:number Tọa độ y của điểm bắt đầu cắt ảnh.
  • width:number Chiều rộng của bản cắt ảnh.
  • height:number Chiều cao của bản cắt ảnh.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var sprite = new Carem.StaticSprite( layer, assetMgr.GetAsset( 'image.jpg' ), 20, 20, 100, 50 );
  • console.log( sprite );

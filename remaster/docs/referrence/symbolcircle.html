Carem.SymbolCircle

Giới thiệu

Carem.SymbolCircle là class dùng để chứa các thành phần dữ liệu, các hàm phương thức nhằm quản lý, và thao tác các dữ liệu liên quan đến hình tròn.

Carem.SymbolCircle được mở rộng với Carem.Graphics và Carem.SceneObject.

Thành phần dữ liệu

DOMCanvasElement

Được dùng để chứa thành phần dom trong html5 được lưu trong thành phần canvas. Tại đây nó được share ra cho Carem.SymbolArc.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolArc( layer );
  • console.log( symbol.DOMCanvasElement );

context

Được dùng để chứa thành phần drawing context của canvas.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolArc( layer );
  • console.log( symbol.context );

SetBackgroundImage( asset, mode )

Được dùng để cài đặt hình nền của hình vẽ bằng hình ảnh.

Tham số

  • asset:object Đối tượng lưu trữ tài liệu.
  • mode:number Hằng số chỉ chế độ repeat.

Các hằng số cho tham số mode

  • CAREM_REPEAT Lặp đều từ ngang đến dọc.
  • CAREM_REPEAT_X Lặp đều theo chiều ngang.
  • CAREM_REPEAT_Y Lặp đều theo chiều dọc.
  • CAREM_NO_REPEAT Chỉ vẽ một lần duy nhất.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolCircle( layer );
  • symbol.SetBackgroundImage( assetMgr.GetAsset( 'image.jpg' ), CAREM_REPEAT );

Giá trị trả về

The new Carem.SymbolCircle instance object.

Draw( ratio, layerX, layerY )

Được sử dụng để render hình vẽ lên bảng vẽ.

Tham số

  • ratio:number Tỉ lệ phóng to hay thu nhỏ.
  • layerX:number Tọa độ x của vị trí layer.
  • layerY:number Tọa độ y của vị trí layer.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolCircle( layer );
  • symbol.Draw( ratio, layerX, layerY );

Carem.SymbolImage

Giới thiệu

Carem.SymbolImage là class dùng để chứa các thành phần dữ liệu, các hàm phương thức nhằm quản lý, và thao tác các dữ liệu liên quan đến hình ảnh.

Carem.SymbolImage được mở rộng với Carem.Graphics và Carem.SceneObject.

Thành phần dữ liệu

DOMCanvasElement

Được dùng để chứa thành phần dom trong html5 được lưu trong thành phần canvas. Tại đây nó được share ra cho Carem.SymbolImage.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolImage( layer );
  • console.log( symbol.DOMCanvasElement );

context

Được dùng để chứa thành phần drawing context của canvas.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolImage( layer );
  • console.log( symbol.context );

image

Được dùng để lưu trữ tài liệu hình ảnh.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolImage( layer );
  • console.log( symbol.image );

idata

Được dùng để lưu trữ dữ liệu pixel của hình ảnh.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolImage( layer );
  • console.log( symbol.idata );

SetStrokeImage( asset, mode )

Được dùng để cài đặt màu viền cho hình vẽ.

Tham số

  • asset:object Đối tượng lưu trữ tài liệu.
  • mode:number Hằng số chỉ chế độ repeat.

Các hằng số cho tham số mode

  • CAREM_REPEAT Lặp đều từ ngang đến dọc.
  • CAREM_REPEAT_X Lặp đều theo chiều ngang.
  • CAREM_REPEAT_Y Lặp đều theo chiều dọc.
  • CAREM_NO_REPEAT Chỉ vẽ một lần duy nhất.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolImage( layer );
  • symbol.SetStrokeImage( assetMgr.GetAsset( 'image.jpg' ), CAREM_REPEAT );

SetStrokeShape( lineJoin, strokeWidth )

Được dùng để cài đặt các hình thức tô viền cho hình vẽ như kiểu gấp, kiểu nối và độ rộng của đường viền hay đường thẳng.

Tham số

Cách sử dụng

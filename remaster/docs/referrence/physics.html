Carem.Physics

Giới thiệu

Carem.Physics là class dùng để chứa các thành phần dữ liệu, các hàm phương thức nhằm quản lý, và thao tác các dữ liệu liên quan đến vật lý.

Thành phần dữ liệu

sceneObject

Được sử dụng để lưu trữ đối tượng sceneObject.

Cách sử dụng

  • var physic = new Carem.Physics( sceneObject );
  • console.log( physic.sceneObject );

velocityX

Được dùng để lưu trữ vận tốc, theo chiều ngang.

Cách sử dụng

  • var physic = new Carem.Physics( sceneObject );
  • console.log( physic.velocityX );

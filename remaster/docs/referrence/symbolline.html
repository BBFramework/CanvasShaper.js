Carem.SymbolLine

Giới thiệu

Carem.SymbolLine chứa các thành phần dữ liệu và phương thức thao tác các đoạn thẳng.

Carem.SymbolLine được mở rộng với Carem.Graphics và Carem.SceneObject

Thành phần dữ liệu

DOMCanvasElement

Được dùng để chứa thành phần dom trong html5 được lưu trong thành phần canvas. Tại đây nó được share ra cho Carem.SymbolLine.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolLine( layer );
  • console.log( symbol.DOMCanvasElement );

context

Được dùng để chứa thành phần drawing context của canvas.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolLine( layer );
  • console.log( symbol.context );

pointArray

Được dùng để lưu trữ các điểm đầu cuối của đoạn thẳng.

Thành phần chỉ đọc. Không khuyến khích sử dụng trực tiếp.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolLine( layer );
  • console.log( symbol.pointArray );

Bạn có thể xem thêm các thành phần dữ liệu từ Carem.Graphics và Carem.SceneObject để có thêm nhiều thông tin.

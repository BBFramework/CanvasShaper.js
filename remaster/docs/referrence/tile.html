Carem.Tile

Giới thiệu

Carem.Tile là class dùng để chứa các thành phần dữ liệu, các hàm phương thức nhằm quản lý, và thao tác các dữ liệu liên quan đến tile.

Carem.Tile được mở rộng với Carem.Graphics, Carem.SceneObject, Carem.Physics và Carem.WorldLimit.

Thành phần dữ liệu

DOMCanvasElement

Được dùng để chứa thành phần dom trong html5 được lưu trong thành phần canvas. Tại đây nó được share ra cho Carem.Tile.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var tile = new Carem.Tile( layer );
  • console.log( tile.DOMCanvasElement );

context

Được dùng để chứa thành phần drawing context của canvas.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var tile = new Carem.Tile( layer );
  • console.log( tile.context );

align

Được dùng để lưu trữ chế độ canh hàng cho tile.

Các hằng số của tham số align

  • CAREM_ALIGN_TOP - 'TOP'
  • CAREM_ALIGN_MIDDLE - 'MIDDLE'
  • CAREM_ALIGN_BOTTOM - 'BOTTOM'
  • CAREM_ALIGN_LEFT - 'LEFT'
  • CAREM_ALIGN_CENTER - 'CENTER'
  • CAREM_ALIGN_RIGHT - 'RIGHT'

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var tile = new Carem.Tile( layer );
  • console.log( tile.align );

order

Được dùng để lưu trữ thứ tự vẽ cho tile.

Các hằng số của tham số order

  • CAREM_ORDER_LEFT - 0
  • CAREM_ORDER_RIGHT - 1

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var tile = new Carem.Tile( layer );
  • console.log( tile.order );

tiles

Được dùng để lưu trữ hình vẽ cho tile.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var tile = new Carem.Tile( layer );
  • console.log( tile.tiles );

AddCell( asset, x, y )

Được dùng để cài đặt độ trong suốt hay chắn sáng của tile.

Tham số

  • asset:number Chế độ loop tile.
  • x:number Tọa độ x của điểm mà muốn vẽ.
  • y:number Tọa độ y của điểm mà muốn vẽ.

  • var layer = new Carem.Layer( 'canvas_id' );

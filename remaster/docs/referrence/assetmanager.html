Carem.AssetManager

Giới thiệu

Carem.AssetManager là class dùng để quản lý các tài nguyên như hình ảnh và âm thanh.

Thành phần dữ liệu

queueType

Một mảng chứa dữ liệu mô tả kiểu dữ liệu tài nguyên.

Cách truy cập

  • var assetMgr = new Carem.AssetManager( '.jpg|.gif|.png' );
  • console.log( assetMgr );

errorCount

Một số nguyên chỉ số lỗi phát sinh trong quá trình tải dữ liệu.

Cách truy cập

  • var assetMgr = new Carem.AssetManager( '.jpg|.gif|.png' );
  • console.log( assetMgr.errorCount );

successCount

Một số nguyên chỉ số thành công trong quá trình tải dữ liệu.

Cách truy cập

  • var assetMgr = new Carem.AssetManager( '.jpg|.gif|.png' );
  • console.log( assetMgr.successCount );

assetList

Một mảng dữ liệu được để lưu trữ tạm thời tài nguyên.

Cách truy cập

  • var assetMgr = new Carem.AssetManager( '.jpg|.gif|.png' );
  • console.log( assetMgr.assetList );

Carem.Graphics

Giới thiệu

Carem.Graphics là class dùng để chứa các thành phần dữ liệu cũng như các hàm phương thức mà quản lý, và thao tác đồ họa. Các kiểu hiển thị (Styling) của hình vẽ (tức là các thành phần đồ họa, đặc biệt là các thành phần hình họa, trong Carem gọi là Symbol) được điều khiển qua Carem.Graphics.

Thành phần dữ liệu

context

Được dùng để lưu trữ đối tượng drawing context của bảng vẽ.

Cách truy cập

  • var graphics = new Carem.Graphics( ctx );
  • console.log( graphics.context );

fillStyle

Được dùng để lưu trữ kiểu tô màu cho hình vẽ, có thể là màu nguyên khối (solid), hay chuyển sắc(gradient), hoặt là mẫu hình ảnh(pattern).

Cách truy cập

  • var graphics = new Carem.Graphics( ctx );
  • console.log( graphics.fillStyle );

patternFill

Được dùng để lưu trữ hình vẽ được tô lên bảng vẽ.

  • console.log( graphics.patternFill );

Được dùng để lưu trữ vòng chuyển sắc lên bảng vẽ.

Cách truy cập

SetMask()

Cài đặt bảng vẽ vào trong trạng thái mask.

Cách sử dụng

  • var rect = new Carem.SymbolRect( layer );
  • rect.SetMask();
  • var roundRect = new Carem.SymbolRoundRect( layer );
  • var arc = new Carem.SymbolArc( layer );
  • arc.ReleaseMask();
  • // Result: 'rect' will mask for 'roundRect', 'arc' will draw normal.

Đối tượng Carem.Graphics hiện tại.

ClearMask()

Hủy trạng thái mask cho bảng vẽ. Hủy ở đây được hiểu là, dừng thôi không mask nữa chứ không phải là thoát ra, hay kết thúc mask.

Cách sử dụng

  • var rect = new Carem.SymbolRect( layer );
  • rect.SetMask();
  • var roundRect = new Carem.SymbolRoundRect( layer );
  • var arc = new Carem.SymbolArc( layer );
  • arc.ClearMask();
  • // Result: 'rect' will mask for 'roundRect', 'arc' will draw normal.

SetShadow( color, offset, blur )

Được dùng để cài đặt bóng đỗ cho hình vẽ.

Tham số

  • color:object Đối tượng chứa dữ liệu màu của bóng đỗ.
  • offset:object Đối tượng chứa khoảng cách của bóng đỗ đến hình vẽ.
  • blur:number Độ lớn của bóng đỗ.

Cách sử dụng

  • var graphics = new Carem.Graphics( ctx );

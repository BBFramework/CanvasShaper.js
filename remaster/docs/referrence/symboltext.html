Carem.SymbolText

Giới thiệu

Carem.SymbolText là class dùng để chứa các thành phần dữ liệu, các hàm phương thức nhằm quản lý, và thao tác các dữ liệu liên quan đến văn bản.

Carem.SymbolText được mở rộng với Carem.Graphics và Carem.SceneObject.

Thành phần dữ liệu

DOMCanvasElement

Được dùng để chứa thành phần dom trong html5 được lưu trong thành phần canvas. Tại đây nó được share ra cho Carem.SymbolText.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolText( layer );
  • console.log( symbol.DOMCanvasElement );

context

Được dùng để chứa thành phần drawing context của canvas.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolText( layer );
  • console.log( symbol.context );

fontName

Được dùng để lưu trữ các điểm đầu cuối của các đường cong hoặt đường thẳng.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolText( layer );
  • console.log( symbol.fontName );

fontSize

Được dùng để lưu trữ kích thước của phông chữ được dùng để tạo text. Giá trị mặc định là 12.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolText( layer );
  • console.log( symbol.fontSize );

textBaseline

Được dùng để lưu trữ thông tin canh hàng dọc của phông chữ được dùng để tạo text.

Giá trị mặc định được sử dụng là giá trị CAREM_TEXT_BASELINE_ALPHABETIC. Ngoài ra, còn có các hằng số giá trị khác như:

  • CAREM_TEXT_BASELINE_TOP - 'top'
  • CAREM_TEXT_BASELINE_BOTTOM - 'bottom'
  • CAREM_TEXT_BASELINE_MIDDLE - 'middle'
  • CAREM_TEXT_BASELINE_ALPHABETIC - 'alphabetic'
  • CAREM_TEXT_BASELINE_HANGING - 'hanging'

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolText( layer );
  • console.log( symbol.textBaseline );

textAlign

Được dùng để lưu trữ thông tin canh hàng ngang của phông chữ được dùng để tạo text.

Giá trị mặc định được sử dụng là giá trị CAREM_TEXT_ALIGN_START. Ngoài ra, còn có các hằng số giá trị khác như:

  • CAREM_TEXT_ALIGN_START - 'start'
  • CAREM_TEXT_ALIGN_END - 'end'
  • CAREM_TEXT_ALIGN_LEFT - 'left'
  • CAREM_TEXT_ALIGN_CENTER - 'center'
  • CAREM_TEXT_ALIGN_RIGHT - 'right'

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolText( layer );
  • console.log( symbol.textAlign );

textList

Được dùng để lưu trữ các tài liệu văn bản mà sẽ được hiển thị.

Thuộc tính textList sẽ lưu trữ các đối tượng văn bản, mà trong đó mỗi đối tượng phải là một json {text: text, x: posX, y: posY, width: textWidth}. posX và posY là các tọa độ điểm của vị trí mà text sẽ được hiển thị. width là kích thước ngang của text được vẽ.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolText( layer );
  • console.log( symbol.textList );

Bạn có thể xem thêm các thành phần dữ liệu từ Carem.Graphics và Carem.SceneObject để có thêm nhiều thông tin.

SetBackgroundImage( asset, mode )

Được dùng để cài đặt hình nền của hình vẽ bằng hình ảnh.

Tham số

  • asset:object Đối tượng lưu trữ tài liệu.
  • mode:number Hằng số chỉ chế độ repeat.

Các hằng số cho tham số mode

  • CAREM_REPEAT Lặp đều từ ngang đến dọc.
  • CAREM_REPEAT_X Lặp đều theo chiều ngang.
  • CAREM_REPEAT_Y Lặp đều theo chiều dọc.
  • CAREM_NO_REPEAT Chỉ vẽ một lần duy nhất.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolText( layer );
  • symbol.SetBackgroundImage( assetMgr.GetAsset( 'image.jpg' ), CAREM_REPEAT );

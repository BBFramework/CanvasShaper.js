Carem.Math

Giới thiệu

Carem.Math là class dùng để chứa các phương thức toán học, Carem.Math là class tĩnh, bạn có thể sử dụng trực tiếp mà không cần tạo cụ thể.

Thành phần dữ liệu

You can see members of Carem.Vec3, Carem.Color for more information.

Thành phần constructor

Carem.Light( x, y, z )

Creates a new Carem.Light instance object.

The params

  • x:number the x-coord of the light position.
  • y:number the y-coord of the light position.
  • z:number the z-coord of the light position.

How to use

  • var light = new Carem.Light( 128.0, 128.0, 128.0 );
  • console.log( light );

The return

The new Carem.Light instance object.

Carem.Layer

Giới thiệu

Carem.Layer chứa các thành phần dữ liệu và phương thức thao tác dữ liệu layer. Layer trong Carem giúp các developer quản lý các tài nguyên của game theo chế độ layer, tức là Layer chứa và hiển thị các hình ảnh, các hoạt hình, hiệu ứng particle.

Thành phần dữ liệu

DOMElement

Được dùng để lưu trữ thành phần DOM của canvas.

Cách truy cập

  • var layer = new Carem.Layer( 'canvas-id' );
  • console.log( layer.DOMElement );

context

Được dùng để lưu trữ thành phần drawing context của canvas.

Cách truy cập

  • var layer = new Carem.Layer( 'canvas-id' );
  • console.log( layer.context );

ratio

Được dùng để lưu trữ thành phần dữ liệu tỉ lệ phóng thu của canvas.

Cách truy cập

  • console.log( layer.ratio );

Được dùng để lưu trữ các thành phần đồ họa của canvas.

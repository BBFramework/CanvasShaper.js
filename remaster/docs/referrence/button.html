Carem.Button

Giới thiệu

Carem.Button là class dùng để chứa các thành phần dữ liệu, các hàm phương thức nhằm quản lý, và thao tác các dữ liệu liên quan đến văn bản.

Carem.Button được mở rộng với Carem.Graphics và Carem.SceneObject.

Thành phần dữ liệu

DOMCanvasElement

Được dùng để chứa thành phần dom trong html5 được lưu trong thành phần canvas. Tại đây nó được share ra cho Carem.Button.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var button = new Carem.Button( layer, assetMgr.GetAsset( 'image.jpg' ), 20, 20, 100, 50 );
  • console.log( button.DOMCanvasElement );

context

Được dùng để chứa thành phần drawing context của canvas.

Cách sử dụng

  • console.log( button.context );

Thành phần constructor

Carem.Button( layer, asset, x, y, width, height )

Tạo mới một đối tượng Carem.Button.

Tham số

  • layer:Carem.Layer Đối tượng layer dùng render hình vẽ.
  • asset:Image Đối tượng hình ảnh.
  • x:number Tọa độ x vị trí hình ảnh.
  • y:number Tọa độ y vị trí hình ảnh.
  • width:number Chiều rộng nút nhấn.
  • height:number Chiều cao nút nhấn.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var button = new Carem.Button( layer, assetMgr.GetAsset( 'image.jpg' ), 20, 20, 100, 50 );
  • console.log( button );

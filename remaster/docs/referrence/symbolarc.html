Carem.SymbolArc

Giới thiệu

Carem.SymbolArc chứa các thành phần dữ liệu và phương thức thao tác các hình vòng cung.

Carem.SymbolArc được mở rộng với Carem.Graphics và Carem.SceneObject

Thành phần dữ liệu

DOMCanvasElement

Được dùng để chứa thành phần dom trong html5 được lưu trong thành phần canvas. Tại đây nó được share ra cho Carem.SymbolArc.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolArc( layer );
  • console.log( symbol.DOMCanvasElement );

context

Được dùng để chứa thành phần drawing context của canvas.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolArc( layer );
  • console.log( symbol.context );

Thành phần constructor

Carem.SymbolArc( layer )

Tạo ra một đối tượng Carem.SymbolArc mới.

Tham số

  • layer:Carem.Layer Layer chứa và vẽ đối tượng.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolArc( layer );
  • console.log( symbol );

Giá trị trả về

The new Carem.SymbolArc instance object.

Các hàm phương thức

SetCircleSlice( startAngle, endAngle )

Được dùng để cài đặt phần cắt của hình vẽ.

Tham số

  • startAngle:number Chỉ số góc bắt đầu cắt.
  • endAngle:number Chỉ số góc kết thúc cắt.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolArc( layer );
  • symbol.SetSize( 50.0, 50.0 );
  • symbol.SetCircleSlice( 45.0, 90.0 );

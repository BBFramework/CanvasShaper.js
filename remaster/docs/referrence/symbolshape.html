Carem.SymbolShape

Giới thiệu

Carem.SymbolShape là class dùng để chứa các thành phần dữ liệu, các hàm phương thức nhằm quản lý, và thao tác các dữ liệu liên quan đến hình họa.

Carem.SymbolShape được mở rộng với Carem.Graphics và Carem.SceneObject.

Thành phần dữ liệu

DOMCanvasElement

Được dùng để chứa thành phần dom trong html5 được lưu trong thành phần canvas. Tại đây nó được share ra cho Carem.SymbolShape.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolShape( layer );
  • console.log( symbol.DOMCanvasElement );

context

Được dùng để chứa thành phần drawing context của canvas.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolShape( layer );
  • console.log( symbol.context );

pointArray

Được dùng để lưu trữ các điểm đầu cuối của các đường cong hoặt đường thẳng.

Thành phần chỉ đọc. Không khuyến khích sử dụng trực tiếp.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolShape( layer );
  • console.log( symbol.pointArray );

openShape

Được dùng để lưu trữ tọa độ điểm, nơi mà sẽ được dùng để bắt đầu vẽ hình họa.

Thành phần chỉ đọc. Không khuyến khích sử dụng trực tiếp.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolShape( layer );
  • console.log( symbol.openShape );

AddPoint( x, y, bX, bY, lX, lY )

Được dùng để thêm các điểm neo và các điểm biên tập vào trong bộ máy vẽ hình họa.

Tham số

  • x:number Tọa độ x của điểm neo hiện tại.
  • y:number Tọa độ y của điểm neo hiện tại.
  • bX:number Tọa độ x của điểm biên tập (thuộc cánh tay phải) của điểm neo kế trước.
  • bY:number Tọa độ y của điểm biên tập (thuộc cánh tay phải) của điểm neo kế trước.
  • lx:number Tọa độ x của điểm biên tập (thuộc cánh tay trái) của điểm hiện tại.
  • ly:number Tọa độ y của điểm biên tập (thuộc cánh tay trái) của điểm hiện tại.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolShape( layer );
  • symbol.AddPoint( 50, 10, 15, 25, 30, 50 );

SetBackgroundImage( asset, mode )

Được dùng để cài đặt hình nền của hình vẽ bằng hình ảnh.

Tham số

  • asset:object Đối tượng lưu trữ tài liệu.
  • mode:number Hằng số chỉ chế độ repeat.

Các hằng số cho tham số mode

  • CAREM_REPEAT Lặp đều từ ngang đến dọc.
  • CAREM_REPEAT_X Lặp đều theo chiều ngang.
  • CAREM_REPEAT_Y Lặp đều theo chiều dọc.
  • CAREM_NO_REPEAT Chỉ vẽ một lần duy nhất.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var symbol = new Carem.SymbolShape( layer );
  • symbol.SetBackgroundImage( assetMgr.GetAsset( 'image.jpg' ), CAREM_REPEAT );

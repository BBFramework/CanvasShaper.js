Carem.Scroller

Giới thiệu

Carem.Scroller là class dùng để chứa các thành phần dữ liệu, các hàm phương thức nhằm quản lý, và thao tác các dữ liệu liên quan đến scroller.

Carem.Scroller được mở rộng với Carem.Graphics, và Carem.SceneObject.

Thành phần dữ liệu

DOMCanvasElement

Được dùng để chứa thành phần dom trong html5 được lưu trong thành phần canvas. Tại đây nó được share ra cho Carem.Scroller.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var scroll = new Carem.Scroller( layer, assetMgr.GetAsset( 'image.jpg' ) );
  • console.log( scroll.DOMCanvasElement );

context

Được dùng để chứa thành phần drawing context của canvas.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var scroll = new Carem.Scroller( layer, assetMgr.GetAsset( 'image.jpg' ) );
  • console.log( scroll.context );

image

Được dùng để lưu trữ tài liệu hình ảnh của scroll.

Cách sử dụng

direct

Được dùng để lưu trữ hướng vận tốc scroll. Giá trị mặc định là {x: 1,y: 0}

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var scroll = new Carem.Scroller( layer, assetMgr.GetAsset( 'image.jpg' ) );
  • console.log( scroll.direct );

Thành phần constructor

Carem.Scroller( layer, asset )

Tạo mới một đối tượng Carem.Scroller.

Tham số

  • layer:Carem.Layer Đối tượng layer dùng render hình vẽ.
  • asset:any Đối tượng tài liệu hình ảnh.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var scroll = new Carem.Scroller( layer, assetMgr.GetAsset( 'image.jpg' ) );
  • console.log( scroll );

Giá trị trả về

Một đối tượng Carem.Scroller mới.

Các hàm phương thức

SetVelocity( value )

Được dùng để cài đặt vận tốc di chuyển của scroll.

Tham số

  • value:number Chỉ số vận tốc của scroll.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var scroll = new Carem.Scroller( layer, assetMgr.GetAsset( 'image.jpg' ) );
  • scroll.SetVelocity( 0.5 );

Giá trị trả về

Đối tượng Carem.Scroller hiện tại.

SetDirect( direction )

Được dùng để cài đặt hướng di chuyển của scroll.

Tham số

  • direction:object Vector hướng di chuyển của scroll.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var scroll = new Carem.Scroller( layer, assetMgr.GetAsset( 'image.jpg' ) );
  • scroll.SetDirect( { x: 10, y: 10 } );

Giá trị trả về

Đối tượng Carem.Scroller hiện tại.

Draw( ratio, layerX, layerY )

Được sử dụng để render scroll lên bảng vẽ.

Tham số

  • ratio:number Tỉ lệ phóng to hay thu nhỏ.
  • layerX:number Tọa độ x của vị trí layer.
  • layerY:number Tọa độ y của vị trí layer.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var scroll = new Carem.Scroller( layer, assetMgr.GetAsset( 'image.jpg' ) );
  • scroll.Draw( ratio, layerX, layerY );

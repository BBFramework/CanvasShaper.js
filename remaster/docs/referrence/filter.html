Carem.Filter

Giới thiệu

Carem.Filter là class dùng để chứa các phương thức filter hình ảnh, Carem.Filter là class tĩnh, bạn có thể sử dụng trực tiếp mà không cần tạo cụ thể.

Các hàm phương thức

Carem.Filter.GrayScale

Được dùng để filter hình ảnh thành màu trắng đen.

Cách sử dụng

  • var image = new Carem.Image( layer, asset );
  • image.SetFilter( Carem.Filter.GrayScale );

Carem.Filter.Convolute( weight, opaque )

Làm nhiễu (noise) hoặt làm sắc nét (sharp) dữ liệu hình ảnh.

Tham số

  • weight:array Mức độ biến đổi.
  • opaque:number Mức độ chắn sáng.

Cách sử dụng

  • var image = new Carem.Image( layer, asset );
  • image.SetFilter( Carem.Filter.Convolute, [ 0, -1, 0, -1, 0, -1, 0, -1, 0 ] );

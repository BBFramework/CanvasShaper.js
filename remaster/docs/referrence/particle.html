Carem.Paticle

Giới thiệu

Carem.Paticle là class dùng để chứa các thành phần dữ liệu, các hàm phương thức nhằm quản lý, và thao tác các dữ liệu liên quan đến particle.

Carem.Paticle được mở rộng với Carem.Graphics, và Carem.SceneObject.

Thành phần dữ liệu

DOMCanvasElement

Được dùng để chứa thành phần dom trong html5 được lưu trong thành phần canvas. Tại đây nó được share ra cho Carem.Paticle.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var particle = new Carem.Paticle( layer, assetMgr.GetAsset( 'image.jpg' ) );
  • console.log( particle.DOMCanvasElement );

Thành phần constructor

Carem.Paticle( layer, asset )

Tạo mới một đối tượng Carem.Paticle.

Tham số

  • layer:Carem.Layer Đối tượng layer dùng render hình vẽ.
  • asset:any Đối tượng tài liệu hình ảnh.
  • density:number Chỉ số mật độ xuất hiện các particle thành phần.

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var particle = new Carem.Paticle( layer, assetMgr.GetAsset( 'image.jpg' ) );
  • console.log( particle );

Giá trị trả về

Một đối tượng Carem.Paticle mới.

Các hàm phương thức

SetMode( mode )

Được dùng để chế độ loop của particle.

Tham số

  • mode:number Chế độ loop particle.

Các hằng số cho chế độ loop:

  • CAREM_LOOP_CYCLE Chế độ lặp về vô cực
  • CAREM_LOOP_KILL Chế độ lặp chỉ một lần

Cách sử dụng

  • var layer = new Carem.Layer( 'canvas_id' );
  • var particle = new Carem.Paticle( layer, assetMgr.GetAsset( 'image.jpg' ) );
  • particle.SetMode( CAREM_LOOP_CYCLE );
